# Pong Game

A classic Pong arcade game built with Python's Turtle graphics library. Challenge a friend in this timeless two-player table tennis simulation!

## Game Description

Pong is a two-dimensional sports game that simulates table tennis. Players control paddles on the left and right sides of the screen, attempting to hit a ball past their opponent's paddle to score points. This implementation features smooth paddle movement, increasing ball speed with each hit, and a clean scoreboard display.

## Features

- **Two-player gameplay** - Perfect for playing with a friend on the same computer
- **Smooth controls** - Responsive paddle movement with keyboard controls
- **Dynamic difficulty** - Ball speed increases with each paddle hit
- **Score tracking** - Clean scoreboard showing both players' scores
- **Collision detection** - Realistic ball physics with wall and paddle collisions
- **Game reset** - Ball resets to center when a point is scored

## Controls

| Player | Action | Key |
|--------|--------|-----|
| Right Paddle | Move Up | ↑ (Up Arrow) |
| Right Paddle | Move Down | ↓ (Down Arrow) |
| Left Paddle | Move Up | W |
| Left Paddle | Move Down | S |
