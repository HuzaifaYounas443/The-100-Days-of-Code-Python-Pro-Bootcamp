# Snake Game (Python Turtle)

A classic Snake Game built using Python's built-in **turtle graphics** module.  
This project is part of my Python learning journey and focuses on object-oriented programming and game logic.

---

## Features

- Smooth snake movement
- Keyboard controls (Up, Down, Left, Right)
- Continuous game loop
- Modular and clean code structure (OOP-based)

---

## Technologies Used

- Python 3.x
- Turtle (built-in graphics library)
- Time module
