#  Snake Game

A classic Snake Game built with Python's Turtle graphics library. Control the snake, eat food, and grow longer while avoiding collisions with walls and yourself!

##  Game Features

- **Classic snake movement** - Smooth control using arrow keys
- **Food mechanics** - Eat food to grow the snake and increase score
- **Collision detection** - Game ends if snake hits the wall or itself
- **Real-time score tracking** - Keep track of your current score and high score
- **Smooth animations** - Fluid movement with screen refresh control
- **Growing mechanic** - Snake grows longer each time it eats food


##  Prerequisites

- Python 3.6 or higher
- Turtle graphics library (comes with Python standard library - no installation required)
